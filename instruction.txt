Перед началом работы установите все необходимые зависимости. Для этого используйте команду: python -m pip install -r requirements.txt

Для запуска программы используйте команду: python run.py

Для начала работы необходимо зарегистрироваться, либо авторизироваться.
Введите в строку-приглашение + для создания нового аккаунта, либо нажмите Enter
и воспользуйтесь тестовым аккаунтом:
Имя пользователя - Test_User
Пароль - 12345

Далее вы можете выбрать одно из предложенных действий:
1. Добавить шутку:
Вы можете ввести свой текст, либо воспользоваться генератором шуток.
Шутка будет сохранена в базе данных и вы сможете просматривать, редактировать,
удалить ее при последующих запусках программы.

2. Смотреть свои шутки:
Просмотр всех шуток в профиле.

3. Смотреть шутку по ID
Вы можете просмотреть свою шутку, ID которой вам известен. ID можно посмотреть в
разделе 2, при просмотре всех шуток. Для Test_User можете воспользоваться ID:
60b7f178173f2c4f85a239fa
60b7fdf3d113e2d3243b9baf
60b7fe01df8796171957528c

4. Обновить текст моей шутки
Выберите свою шутку указав ID. Содержание можно изменить на собственное, или
сгенерировать автоматически.

5. Удалить мою шутку
Выберите свою шутку указав ID. Функция удалит шутку безвозвратно.

Программа использует для хранения данных MongoDB Atlas.
Для генерации шуток используется geek-jokes.
